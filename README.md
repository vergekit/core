# @vergekit/core

Core runtime helpers for Verge Kit applications.

## Exports

- `@vergekit/core/utilities`
- `@vergekit/core/http`
- `@vergekit/core/email`
- `@vergekit/core/auth`

## Development

```bash
npm install
npm run verify
```

Build before publishing:

```bash
npm run build
npm publish --access public
```
